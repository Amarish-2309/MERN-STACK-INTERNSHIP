HTML

HyperText Markup Language is the standard markup language for creating web pages. It uses tags to define the structure, content, and layout of a webpage, including text, images, links, and other elements.

CSS

Cascading Style Sheets is a styling language used to describe the presentation of a document written in HTML, allowing you to control the look and feel of web pages. It allows you to add color and animations.

BS

Bootstrap is a popular front-end framework that provides pre-designed and reusable components, as well as a responsive grid system, to help build responsive and visually appealing websites.

JS

JavaScript is a versatile programming language that enables interactive and dynamic functionality on web pages, allowing for behaviors such as user interactions, data manipulation, and dynamic content updates.
